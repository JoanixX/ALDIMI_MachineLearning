# ALDIMI Predict - Dataset Sintético v2 Ampliado

Dataset sintético ampliado para el proyecto grupal de Machine Learning.

## Tamaño
- Pacientes/riesgo: 10,000 registros
- Inventario diario: 18,000 registros
- Ítems: 30 productos
- Donaciones: 1,500 registros
- Capturas IA/OCR: 5,000 registros

## Uso
Sirve para EDA, limpieza, feature engineering, clasificación de prioridad, predicción de consumo y alertas de stock crítico a 7 y 14 días.
